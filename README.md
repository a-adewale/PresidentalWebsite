# State House Static Website

Open `index.html` in a browser.

This version keeps the site static and simple:

- One CSS file: `main.css`
- One JavaScript file: `scripts.js`
- CSS-only mobile menu
- CSS-only search panel
- CSS-only tools panel
- CSS-controlled hero arrows, dots, fade and zoom-out animation
- JavaScript only for the live clock, font size class, hero auto-advance, accordions, contact demo and static search results.
